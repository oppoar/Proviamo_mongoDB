# Proviamo_mongoDB
Progetto con repository di immagini mongo db


Testato con: 
Angular v19
node x21 o maggiore




## Installazione

```
git clone https://github.com/oppoar/Proviamo_mongoDB.git

cd mongoDbRepository

npm install

cd backend

npm install
```


## Avvio

```
cd backend

node server.js

cd ..

ng serve
```
